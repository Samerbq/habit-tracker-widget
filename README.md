habit tracker widget

A 90 day habit tracker widget for IOS ( runs using the Scriptable app ) 

why :
habbit tracker app wants you to pay a monthly ( or one time . . ) subscription just to show dots on a screen . .

You do not need that. With this script and the free Scriptable app you get the exact same thing.

( for free and you own the code entirely ) 

what it does :
displays a grid of dots - one per day of your 90-day challenge. Filled dot = a day done. 

A footer shows your challenge name and days remaining.

requirements 
Iphone with Scriptable app ( free on the App Store )


setup :
download Habit Tracker.js and open it in Scriptable
1. edit the config at the top of the file
2. change the challenge name and start date 
3. add a Scriptable widget to your home screen and select this script
